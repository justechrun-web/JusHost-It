# **App Name**: JusHostIt Dashboard

## Core Features:

- Site Creation: Create a new hosting site with a domain and selected hosting plan.
- Usage Monitoring: View resource usage such as storage, bandwidth, and uptime for each site.
- Billing Management: Manage subscription, view billing history, and update payment information.
- Domain Suggestions: Suggest available domain names to improve new domain creation.
- Ticket support: Open support tickets.
- Login-ready Feature: Enable customers to login, store relevant account details using authentication. When logged in customers may manage sites and access details.

## Style Guidelines:

- Primary color: Dark slate blue (#374151) to create a professional and trustworthy feel.
- Background color: Very light gray (#F9FAFB), almost white, providing a clean and modern look.
- Accent color: Sky blue (#0EA5E9) for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern and neutral feel.
- Use clear, simple icons from Lucide to represent different hosting features and services.
- Use a sidebar navigation for main sections (Dashboard, Sites, Billing, Support) and a clean, card-based layout for data display.
- Subtle transitions and animations on button clicks and data updates to improve user experience.
